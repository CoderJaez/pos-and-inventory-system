# Basic User Guide for Store Staff

## Sell items
1. Tap **Start Selling**.
2. Tap products to add to cart.
3. Adjust quantity/price if needed.
4. Choose **Cash** or **GCash**.
5. Tap **Complete Sale**.

## Inventory
1. Tap **Inventory**.
2. Tap **Add Product** for new item.
3. Use stock adjust controls for restock or stock out.

## Offline use
- If internet is down, checkout is saved locally.
- Once online, queued sales auto-sync.
- Check online/offline badge on home screen.
