# PWA Installation Guide

## Android (Chrome)
1. Open app URL.
2. Tap browser menu.
3. Tap **Install app**.
4. Confirm install.

## Desktop (Chrome/Edge)
1. Open app URL.
2. Click install icon in address bar.
3. Confirm install.

## Verify
- App opens in standalone window.
- `/offline` fallback appears when network is disabled.
